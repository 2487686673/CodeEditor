# Syntax-Highlighting-Editor
移植代码笔记的语法高亮编辑器<br>

语法高亮编辑器是一款高效、稳定的代码编辑器。<br>
此编辑器基于Textwarrior二次开发，采用Java词法分析器编写而成，拥有高性能、代码智能提示补全等特点！<br>

语法着色<br>
行号显示<br>
代码智能提示补全<br>
高性能，超过10000行时流畅编辑<br>
撤销重做<br>
一键注释<br>
高效的选择和编辑功能<br>
